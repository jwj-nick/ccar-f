# CCAR-F Study

**Claude Certified Architect – Foundations (CCA-F)** 학습 지도 — 5개 도메인의 *무엇을 공부하고, 무엇을 실습하고, 어떤 공식·무료 자료로 파고들지*를 한 곳에.

> ⚠️ **비공식** 개인 learn-in-public 프로젝트. Anthropic와 무관. 사실 근거는 [docs.claude.com](https://docs.claude.com) · [Anthropic Academy](https://anthropic.skilljar.com). 초안(v0.1) — 공부가 진행되며 계속 정교화됩니다.

## 무엇
- **개요** — 시험 팩트, 도메인 비중, 공식 학습 경로(Academy 7코스), 학습 루프, 6개 시나리오, 10주 일정.
- **도메인 D1~D5** — 핵심 개념 · 헷갈리는 용어 · 시험 판단 포인트 · 흔한 오답 · 워크드 시나리오 · 자가진단 · 공부/실습 · 자료.
- **자료** — 공식(Academy·docs·engineering 글) + 무료(WikiDocs 실전 대비서·GitHub 다국어 가이드).

## 기술
바닐라 HTML/CSS/JS, 빌드 없음, 폰 우선, 라이트/다크. 데이터는 `content.js`에 분리 — 학습 진척 시 이 파일만 갱신.

```
index.html · styles.css · app.js · content.js
```

## 로컬 실행
정적 파일이라 `index.html`을 브라우저로 열면 됨. (또는 `python -m http.server`)

## 라이선스 / 저작권
공식 코스 원문·시험 문항은 복제하지 않고 공식 링크로 연결. 콘텐츠는 공개 소스에서 증류한 요약.
